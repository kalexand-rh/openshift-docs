= Adding compute machines to AWS by using CloudFormation templates
include::modules/common-attributes.adoc[]
:context: adding-aws-compute-user-infra

toc::[]

You can add more compute machines to your {product-title} cluster on
Amazon Web Services (AWS) that you
created by using the sample CloudFormation templates.

.Prerequisites

* You installed your cluster on AWS by using the provided
xref:../../installing/installing_aws/installing-aws-user-infra.adoc#installing-aws-user-infra[AWS CloudFormation templates].
* You have the JSON file and CloudFormation template that you used to create the
worker machines during cluster installation. If you do not have these files,
you must recreate them by following the instructions in the
xref:../../installing/installing_aws/installing-aws-user-infra.adoc#installing-aws-user-infra[installation procedure].
