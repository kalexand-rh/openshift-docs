include::modules/common-attributes.adoc[]

You can remove a cluster that you deployed to Amazon Web Services (AWS) by using
CloudFormation templates.

.Prerequisites

* You installed a cluster by using the provided
xref:../../installing/installing_aws_user_infra/installing-aws-user-infra.adoc#installing-aws-user-infra[AWS CloudFormation templates].
